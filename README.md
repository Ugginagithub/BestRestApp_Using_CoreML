"BestRestApp"

This app is mainly about the predicting the sleeping hours of a person, based on his/her habits.

We will take inputs from the user and we will predict the sleeping hours,

This app is simple, but we can understand how to use the coreML in our app, I have build an ML model using CoreML, the powerful framework in IOS,
I've trained the model using sample CSV file data. This app helps to understand about coreML very well, we can integrate great model in our app easily.
